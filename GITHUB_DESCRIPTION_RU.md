# Library Orchestrator (LibSort)

**Library Orchestrator** (или **LibSort**) — это локальное приложение с графическим интерфейсом (GUI) для интеллектуальной сортировки, дедупликации и нормализации больших книжных коллекций. 

Приложение сканирует исходные папки, рекурсивно распаковывает архивы, определяет форматы файлов (отсекая мусор), находит дубликаты по содержимому книг (хэш `XXH64`), извлекает метаданные и автоматически переносит уникальные книги в структурированную библиотеку по жанрам и авторам.

---

## Итоговый формат библиотеки

```text
TARGET_DIR\Жанр\Первая буква автора\Автор\Книга.zip
```

Каждая книга упаковывается в индивидуальный архив через 7-Zip. Дубликаты и не-книжные файлы изолируются в папки `Duplicates` и `NoBook` внутри целевой директории.

---

## Ключевые особенности

1. **Многопоточный конвейер (Pipeline)**: Процесс разбит на независимые стадии с собственными очередями и пулами воркеров для оптимизации нагрузки на дисковую подсистему и процессор.
2. **Локальный ИИ (Ollama)**: Поддержка интеграции с локальными LLM (например, через LM Studio) для автоматического извлечения авторов, названий и жанров, если имя файла неинформативно.
3. **Поиск по ISBN**: Автоматическое извлечение кодов ISBN и запрос метаданных через внешние API (Open Library) с локальным кэшированием запросов в SQLite.
4. **Умное распознавание книг (Агент A3)**: Проверка сигнатур файлов (`magic bytes`), анализ книжных расширений и сканирование текстовых файлов на ключевые токены (главы, копирайты, прологи).
5. **Интуитивный GUI (Tkinter)**: Живой дашборд со счетчиками прогресса, логом событий, статусами загрузки очередей каждого агента и вкладкой детальных настроек.
6. **Возобновляемость и безопасность**: Возможность остановить конвейер в любой момент без повреждения данных и продолжить работу позже.

---

## Архитектура агентов (Stage-based Pipeline)

Логика обработки полностью разделена на изолированные модули в директории `agents/`:

* **A1 Поиск (`agent_a1_search.py`)** — Сканирование источников и регистрация файлов.
* **A2 Распаковка (`agent_a2_unpack.py`)** — Рекурсивное извлечение архивов через 7-Zip во временную папку.
* **A3 Книга? (`agent_a3_detect.py`)** — Фильтрация мусора, проверка сигнатур и структуры книг.
* **A4 XXH64 (`agent_a4_dedupe.py`)** — Быстрый расчет хэшей содержимого для точной дедупликации.
* **A5 Теги (`agent_a5_tags.py`)** — Парсинг встроенных метаданных (FB2, EPUB, PDF).
* **A5b ISBN (`agent_a5b_isbn.py`)** — Поиск ISBN кодов и обогащение метаданных.
* **A6 Ollama (`agent_a6_lm.py`)** — Нейросетевой анализ сложных названий и жанров.
* **A7 Переименование (`agent_a7_rename.py`)** — Формирование финальных имен и путей с опциональным переводом.
* **A8 Упаковка (`agent_a8_pack.py`)** — Финальное сжатие книг и интеграция в целевую библиотеку.

## Быстрый запуск

1. Убедитесь, что на компьютере установлены **Python 3.10+** и **7-Zip** (утилита `7z.exe` должна быть доступна в путях системы).
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите графический интерфейс:
   ```bash
   python run.py
   ```
4. Укажите папки источников (`SOURCE_DIRS`), целевую папку библиотеки (`TARGET_DIR`) и запустите процесс кнопкой **Старт**.
